## ğŸ¯ Getting Started

### Prerequisites
- A modern web browser (Chrome, Firefox, Safari, or Edge)
- No installation or dependencies required!

### Usage

1. **Clone the repository**:
   ```bash
   git clone https://github.com/shubham1328/dsa-animation.git
   cd dsa-animation
   ```

2. **Open in browser**:
   ```bash
   # Simply open index.html in your browser
   open index.html
   # Or on Linux
   xdg-open index.html
   # Or on Windows
   start index.html
   ```

3. **Or use a local server**:
   ```bash
   # Using Python 3
   python -m http.server 8000
   
   # Using Node.js with http-server
   npx http-server
   ```

4. **Navigate to**: `http://localhost:8000`

## ğŸ“– How to Use

### Learning Topics
1. Navigate to the **Topics** section
2. Click on any topic card to explore
3. Each topic includes explanations and related problems

### Algorithm Visualization
1. Go to the **Visualizer** section
2. Select an algorithm from the dropdown menu
3. Adjust the speed slider to control animation speed
4. Click **Start** to begin the visualization
5. Watch the algorithm execute with real-time statistics
6. Use **Reset** to generate a new random array

### Practice Problems
1. Visit the **Practice** section
2. Filter problems by difficulty (Easy, Medium, Hard)
3. Click **Solve** on any problem
4. Read the problem description and examples
5. Write your solution in the code editor
6. Click **Run Code** to test or **Submit** for final submission
